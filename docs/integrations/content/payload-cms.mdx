# Payload CMS Integration

Payload CMS is now integrated into our main [Content Management guide](/docs/content-management). You'll find everything you need to get started with structured content management.

## Quick Setup

```bash
DATABASE_URL=postgresql://user:pass@localhost:5432/yourdb
PAYLOAD_SECRET=your_secret_key_here
```

- ✅ Powerful admin interface
- ✅ Custom content types and fields
- ✅ User management and roles
- ✅ Media library for uploads

- ✅ Real-time editing
